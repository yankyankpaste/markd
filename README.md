## Getting Started

Okey doke,  
 
### Installation

SImples

1. Get a free API Key at [https://example.com](https://example.com)
1. Clone the repo
   ```sh
   git clone https://github.com/yankyankpaste/bookmarker.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
3. Run locally
   ```js
   npm start;
   ```
